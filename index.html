Front-End web development...

Front-end web development, also known as client-side development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly. The challenge associated with front end development is that the tools and techniques used to create the front end of a website change constantly and so the developer needs to constantly be aware of how the field is developing. The objective of designing a site is to ensure that when the users open up the site they see the information in a format that is easy to read and relevant. This is further complicated by the fact that users now use a large variety of devices with varying screen sizes and resolutions thus forcing the designer to take into consideration these aspects when designing the site.

Front-End Jobs Titles

The common, or most used (i.e., generic), title for a front-end developer is, "front-end developer" or "front-end engineer". Note that any job that contains the word "front-end", "client-side", "web UI", "HTML", "CSS", or "JavaScript" typically infers that a person has some degree of HTML, CSS, DOM, and JavaScript professional know how

jQuery

Licensed under MIT, jQuery — the JavaScript library was designed for the simplification of HTML DOM tree traversal and manipulation. It is also used for event handling, Ajax, and CSS animation..

Git

Git is the leading distributed version control system used for tracking changes. It is written in Shell, C, Perl, and Tcl and is open-source. The basic premise for its design is simplicity, speed, full distribution for non-linear workflows, integrity and support for multiple parallel branches. .

Angular.js

Angular.js is an extensible JavaScript-based open-source front-end development tool and web framework. Its chief goal includes smooth and easy development and testing of apps. It also provides client-side model–view–controller (MVC) and model–view–view-model (MVVM) architecture.

HTML5 Boilerplate

HTML5 Boilerplate is a CSS, HTML and JavaScript-based template that is used for the creation of cross-browser compatible websites. It comes with HTML5 ready elements and features that aid in building a comprehensive and robust website.
